# Searching-for-red-giants-from-J-PLUS-images-using-self-supervised-learning
## the Catalog of Red Giant Candidates
The StellarLoMaR column provides the probability predicted by the StellarLoMaR model that a given source is a red giant star.  
The isTrain flag indicates whether the object was included in the training sample of the model.  
The High_confidence flag specifies whether the candidate satisfies the high-confidence selection criteria adopted in this study.  
The Source column records the origin of each candidate: New denotes candidates newly identified in this work; Simbad indicates sources that have been independently confirmed and classified as red giant branch (RGB) or asymptotic giant branch (AGB) stars in the SIMBAD database; Wu refers to objects drawn from the red giant catalog of Wu et al. (2019); and Anders denotes sources taken from the catalog of Anders et al. (2023).  
The parameters Teff, log g, and [Fe/H] are sourced from LAMOST and are provided only for stars with corresponding records in the LAMOST database.  
The corresponding Teff,pre, log gpre, and [Fe/H]pre represent regression-based estimates derived from J-PLUS multi-band photometric data and are intended to provide preliminary physical parameter references for stars lacking spectroscopic observations.
